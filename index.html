Step 1

Find a bootstrap template that you like by going to Start Bootstraps website, and select a free theme.

Step 2

Click the button "View on Github" in the side menu. This will take you to Github where you can than fork the template.

Step 3

Go into settings and change the name of your repository to (profile name).github.io. This will allow your site to be active and viewable.

Step 4

Open up Visual Studio Code, and go file > open... & open up the folder where you will keep your project stored in. Once you've opened up the folder, click on terminal > new terminal. Make sure your terminal default shell is on Git Bash. Then Clone your repository and follow the steps in the image to the left to begin being able to edit your site.

Step 5

Once you are able to push items to your repository, you'll be able to go in and start making changes to the template. I ended up changng all the data to information about myself, including changing out the profile picture by adding my own image to the img folder and linking to my profile picture instead of the templates. Any time you make a change tho, make sure to add a commit, and push the changes to the orgin master.

Step 6

I ended up adding in a new section for projects. To do so I copied a chunk of section code, and took out everything not needed for projects, leaving just an H1 tag and paragraph tag. Also make sure to add a new nav item, so you're able to scroll to the section.

Step 7

One you've made basic edit changes you can go into the resume.css and change the colors. This will make your profile page more personable by adding the colors you like. You can also go in and change any other styling if you so please. Also make sure to add the proper ref in the head, as it won't be able to read your color chnages if you don't do that.

Step 8

Once you've made all the changes your heart desires you'll have your final product, a great looking profile page that can be used for resumes! Congrats!
